# Sundance Water

Source code for the Sundance Water (a Kinetico Water Systems dealer) website.

Specs:
  * Node.js
  * React
  * ES6 - compiled with Babel   
  * Webpack
  * SCSS

To install / run / buil:
  * $ npm install
  * $ npm start
  * visit localhost:8080
  * $ npm build
